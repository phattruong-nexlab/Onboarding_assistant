# Change Log

> Breaking changes, API changes, DB schema changes quan trọng.
> Đọc file này trước khi update dependency hoặc migrate.

---

## Format

```
## [YYYY-MM-DD] — [Feature/Release name]

### Breaking Changes
- [Thay đổi nào sẽ break existing code]

### API Changes
- Added: [Endpoint mới]
- Modified: [Endpoint thay đổi — old vs new]
- Deprecated: [Endpoint sắp bỏ]
- Removed: [Endpoint đã bỏ]

### Database Changes
- [Migration nào đã chạy]

### Config Changes
- [Env var mới hoặc thay đổi]
```

---

## [YYYY-MM-DD] — Project Init

### Notes
- Project khởi tạo với SDD package structure
- Chưa có breaking changes
